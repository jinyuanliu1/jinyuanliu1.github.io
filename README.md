# Jinyuan Liu — personal homepage

This is a deliberately small, single-page Jekyll site hosted by GitHub Pages.

## Updating the site

- Biography, research interests, education, and links: edit `index.html`.
- Publications: edit `_data/publications.yml`; copy an existing entry and place the newest item at the top.
- Profile photo: replace `images/jinyuan.png` while keeping the same filename.
- CV: replace `files/JinyuanLiu_s_Resume.pdf` while keeping the same filename.
- Styling: edit `assets/css/site.css`.

GitHub Pages rebuilds the site automatically after changes are pushed to the default branch.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

Then open <http://localhost:4000>.
